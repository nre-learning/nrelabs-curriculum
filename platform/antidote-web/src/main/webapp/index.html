Automation. Un-shackled.

NRE Labs is a no-strings-attached initiative to make automation more consumable to the masses. Through short exercises, totally contained within the browser, you can learn the skills that will put you on the path to becoming a Network Reliability Engineer

Get started learning about automation, right now, all in your browser!

No experts allowed.

NRE Labs assumes nothing about your background or training. Everything is contained in the browser, and lets you learn at your own pace.

Paywalls are so last year.

We don't track you for sales purposes. This is an educational tool aimed at helping the industry consume automation more readily.

Open Source.

NRE Labs is built on Antidote, an open source project published on Github under an Apache v2 license to make it easier to consume bite-sized chunks of automation.
